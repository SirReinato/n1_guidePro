Resetar Senha de Usuários no Active Directory

Este procedimento orienta a equipe de atendimento remoto N1 a resetar a senha dos usuários no AD - MDS.

1 - Abrir Conexão de Área de Trabalho Remota

Conecte-se ao servidor usando a seguinte IP: 10.222.11.65.

2 - Pesquisar Active Directory Users and Computers

No menu iniciar, pesquise por 'Active Directory Users and Computers'.

3 - Localizar o Usuário

Pesquise pelo nome do usuário cuja senha precisa ser resetada.

4 - Resetar Senha

Clique com o botão direito do mouse no nome do usuário e selecione 'RESET PASSWORD'.

5 - Inserir Senha Padrão

Insira a senha padrão: 'mds@123456' para usuários do MDS ou 'mesp@123456' para usuários do esporte. A nova senha deve começar com letra maiúscula, conter 10 caracteres, números e caracteres especiais, e não pode conter o nome do usuário.
